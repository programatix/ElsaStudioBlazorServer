# ElsaStudioBlazorServer


## Running the servers

Run the server
```
dotnet run --project ElsaServer --urls "https://localhost:5001"
```

Run the studio
```
dotnet run --project ElsaStudioBlazorServer --urls "https://localhost:7001"
```
